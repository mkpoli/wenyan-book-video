# Processor

1. Use `segment-text.py` to segment the text into smaller chunks to `segments/`
2. Use `https://qieyun-tts.com/home` to generate the audio for each chunk and save into `audios/`
3. Use `audio-femalize.py` to femalize the audio into `audios/female/`
4. Use `video-renderer.py` to render the video into `videos/`
